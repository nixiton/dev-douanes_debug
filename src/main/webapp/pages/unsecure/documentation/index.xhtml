Administration et Opérations

La page par défaut de l'application dans cette comptabilité est le tableau de bord ou Administration, cette menu est disponible pour toute utilisateur quelque soit sa fonction.

Pour chaque utilisateur, ce tableau présente la liste des opérations effectuées par l'utilisateur courant. Les détails sur les opérations effectués figurant sur le tableau sont :

  • Date : date de l'opération
  • Poste : poste de l'opérateur
  • L'état : Etat actuel de l'opération, il existe 4 type d'état
    • ACCEPTEE
    • EN ATTENTE
    • REFUSEE
  • Opérateur : Immatriculation de l'opérateur
  • Type d'Opération : Le type d'opération éfféctuée, il existe 7 types d'opérations disponibles :
    • Entrée matériel
    • Sortie matériel
    • Saisie des référentiels
    • Attribution
    • Détachement
    • Entrée Article
    • Sortie Article
  • Pièces justificatives : lien pour générer un fichier concernant la justification de l'opération effectuée.

Des filtres sur l'état et le type d'opération sont disponibles, mais également un filtre pour les recherches globales.

Saisie des référentiels

Fonctionnalité destinées seulement pour les personnels du Service Informatique (SI) et Service Equipement (SE) pour enregistrer les référentiels utilisés dans les opérations à effectuer. L'interface générale de cette menu comprend tous les tableaux contenant les différentes référentiels utilisés pour les opérations dans cette application.

Chaque tableau de référentiels contient un bouton avec lequel l'utilisateur peut accéder à l'interface d'ajout de nouvelle type de référentiel.
Chaque tableau contient également la liste de référentiels d'une type de référentiels choisi (ex: liste de nomenclature ou liste de type de matériels).
Dans chaque tableau, l'utilisateur courant peut modifier ou supprimer des référentiels.

Un filtre est également disponible en cliquent sur le bouton cela facilite à l'utilisateur, la recherche de référentiel

N.B : Aucun doublon n'est autorisé dans les saisies des référentiels. Un message d'erreur comme ci-dessous devrait apparaître en cas de doublon detecté.

Ci-dessous les différentes référentiels utilisés dans les opérations dont l'utilisateur courant peut ajouter :

  • Nomenclature

      • Cette interface est utilisée pour ajouter une nouvelle nomenclature, 2 champs sont à remplir :

    • Désignation : texte définissant la nomenclature ex: Combustibles et Luminaires
    • Nomenclature : code unique selon la nomenclature officielle ex:1

    Tous ces champs sont obigatoires, une message d'erreur apparaît dans le cas contraire

  • Type de matériels

      • Cette interface est utilisé pour ajouter un nouveau type de materiel, 3 champs sont à remplir :

    • Désignation : texte définissant le type de matériel ex:Lampe tube fluorescent
    • Code type matériel : code unique et de 3 lettres capitales et une chiffre dont les 3 premières lettres sont les 3 premières de la designation et le chiffre incrémenter pour les matériles ayant le même type. ex: LAM1
    • Catégorie : Catégorie de la nomenclature associé

    Tous ces champs sont obigatoires, une message d'erreur apparaît dans le cas contraire

  • Etat de matériels

      • Cette interface est utilisé pour ajouter un nouveau type d'etat de matériel

    • Désignation : texte définissant l'état du matériel

    Ce champ est obigatoire, une message d'erreur apparaît dans le cas contraire

  • Agents (Utilisateurs)

    • Cette interface est utilisé pour créer une nouvelle utilisateur, 7 champs sont à remplir :

    • Nom et prénom : Champ de type lettre, contenant le nom et prénom du nouveau agent
    • Immatriculation : Champ de type numérique uniquement, contenant la matricule du nouveau agent tel que la longueur minimal du matricule est 6
    • Fonction : Menu déroulant les différentes fonctions existantes
    • Mot de passe : Champ de type AlphaNumeric, contenant le mot de passe de l'agent tel la longueur minimale du mot de passe est 8
    • Retaper mot de passe : Champ de type AlphaNumeric, contenant le mot de passe identique au champ au précédent
    • Direction : Menu déroulant les différentes direction existantes
    • Poste : Menu déroulant les différentes poste existantes

    Tous ces champs sont obigatoires, une message d'erreur apparaît dans le cas contraire

  • Marques

      • Cette interface est utilisé pour ajouter une nouvelle marque

    • Désignation : texte définissant la marque

    Ce champ est obligatoire, une message d'erreur apparaît dans le cas contraire

    SUPPRIMER CETTE SECTION FONCTION

  • Fonction

      • Cette interface est utilisé pour ajouter une nouvelle marque

    • Désignation : texte définissant la nouvelle fonction
    • Fonction : texte (unique) définissant le rôle associé à cette fonction (ex: role_admin/role_sise/role_dc/role_gac/role_agent)

    Tous ces champs sont obigatoires, une message d'erreur apparaît dans le cas contraire

  • Motifs d'aquisitions

      • Cette interface est utilisé pour ajouter une nouvelle motif d'acquisitions de matériels ou équipements

    • Désignation : texte définissant le motif d'acquisition

  • Motifs de décharge

      • Cette interface est utilisé pour ajouter une nouvelle motif de décharge

    • Désignation : texte définissant le motif de décharge

    Ce champ est obigatoire, une message d'erreur apparaît dans le cas contraire

  • Financement

      • Cette interface est utilisé pour ajouter une nouvelle source de financement

    • Désignation : texte définissant le financement

    Ce champ est obigatoire, une message d'erreur apparaît dans le cas contraire

  • Fournisseurs

      • Cette interface est utilisé pour ajouter une nouvelle fournisseur

    • Désignation : texte définissant le fournisseur

    Ce champ est obigatoire, une message d'erreur apparaît dans le cas contraire

  • Poste

      • Cette interface est utilisé pour ajouter une nouvelle poste

    • Désignation : texte définissant la nouvelle poste

    Ce champ est obigatoire, une message d'erreur apparaît dans le cas contraire

  • Direction

      • Cette interface est utilisé pour ajouter une nouvelle direction

    • Désignation : texte définissant la nouvelle direction
    • Code direction : code unique représentant la direction
    • Intitulé : Intitulé de la direction qui peut changer au fur et mésure d'une nouvelle assignation
    • Localité : Localisation de la direction
    • Budget : Budget correspondant à la direction - utilisée dans les états réglementaires Ajouter lien vers cette section
    • Subdivision du chapitre :Subdivision du chapitre correspondant à la direction - utilisée dans les états réglementaires Ajouter lien vers cette section

  • Motif de sortie

      • Cette interface est utilisé pour ajouter une nouvelle motif de sortie

    • Désignation : texte définissant le motif de sortie (ex: Affection/Perte/Départ)

    Ce champ est obigatoire, une message d'erreur apparaît dans le cas contraire

  • Devise

      • Cette interface est utilisé pour ajouter une nouvelle devise de la république

    • Désignation : texte définissant la nouvelle devise de la république
    • Date du changement : Date définissant le debut de l'utilisation de la nouvelle devise de la république

    Tous ces champs sont obigatoire, une message d'erreur apparaît dans le cas contraire

Prise en charge

Cette menu est uniquement destinées au Dépositaire Comptable (DC) pour pouvoir visualiser ou modifier les matériels prise en charge dans une facture. L'utilisateur peut également ajouter les nouveaux matériaux à prendre en charges à partir de la facture reçue. Cette prise en charge devra être validée par le GAC

Seuls les utilisateurs Service Equipement (SE) ,Service Informatique (SI) ont accés à ce menu pour la modification des données des matériels qui ont été déjà validés par le GAC de la direction

L'interface de ce menu contient 2 tableau dont la liste des matétriels prise en charge acceptée et la liste des matériels prise en charge non acceptée

Concernant les détails, les modifications, et l'ajout, les interfaces sont similaire à ceux du saisie des matériels existants mais des champs supplémentaires sont à remplir et ces matériels nécessitent une validation.

Attribution

Cette menu est uniquement destinées au Dépositaire Comptable (DC) pour pouvoir attribuer un matériel à un agent (détenteur) de sa diréction.

Pour attribuer un matériel à un agent, les champs(obligatoires) suivants sont à remplir :

    Matériel : liste déroulante des matériel disponibles (à partir de son numéro de série)

    Nomenclature : Rempli automatiquement selon le matériel choisi

    Marque : Rempli automatiquement selon le matériel choisi

    Référence : Rempli automatiquement selon le matériel choisi

    Détenteur : Liste déroulante des agents pour être le nouveau détenteur du matériel choisi.

    Nom : Rempli automatiquement selon le détenteur choisi

  • La soumission de l'opération d'attribution aux GAC de la Direction se fait en cliquant sur le boutton Attribuer

    • N.B : Les champs Matériel et Détenteur sont obligatoires, une message d'erreur apparaît dans le cas contraire

Détachement

Cette menu est uniquement destinées au Dépositaire Comptable (DC) pour pouvoir détacher un matériel détenu par un agent de sa direction.

Pour détacher un matériel détenu par un agent, les champs(obligatoires) suivants sont à remplir :

    Détenteur : Liste déroulante des agents (représenté par sa matriucle) détenteur de matériel.

    Nom : Rempli automatiquement selon le détenteur choisi

    Matériel : Liste déroulante des matériel disponibles pour pouvoir sélectionner le matériel retenu

    Marque : Rempli automatiquement selon le matériel choisi

    Référence : Rempli automatiquement selon le matériel choisi

    Codification : Rempli automatiquement selon le matériel choisi

    Motif : Liste déroulante des différentes motifs de détachement

  • La soumission du détachement aux validateurs (GAC) se fait en cliquant sur le bouton Détacher

    • N.B : Les champs Détenteur, Matériel et Motif sont obligatoires, une message d'erreur apparaît dans le cas contraire

Sortie de matériel (décharge)

Cette menu est uniquement destinées au Dépositaire Comptable (DC) pour pouvoir décharger ou sortir un matériel détenu selon le motif.

Pour procéder à une sortie de matériels vers une autre Localité/Direction, les champs(obligatoires) suivants sont à remplir :

    Motif de sortie : Liste déroulante des choix possibles de motif de sortie d'un matériel

    Matériel : liste déroulante des matériel à sortir

    Marque : Rempli automatiquement selon le matériel choisi

    Référence : Rempli automatiquement selon le matériel choisi

    Numéro de série : Rempli automatiquement selon le matériel choisi

  • La soumission de la décharge aux validateurs (GAC) se fait en cliquant le bouton Décharger

    • N.B : Le champ Matériel est obligatoire, une message d'erreur apparaît dans le cas contraire

Traçabilité

Cette menu est accéssible pour les personnels du Gestionnaires d'Activités (GAC), Services Equipements (SE),Service Informatique (SI), Direction Générale des Douanes (DGD) et Direction des Ressources et de la Formation (DRF) .
